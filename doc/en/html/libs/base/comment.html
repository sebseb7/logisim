Comment Tool

Library: Mouse Tools
Introduced: 2.0 Beta 1

Behavior

The Comment Tool adds free-form text comments to a circuit. It is a limited and less useful version of the Text Tool, as it can't edit labels on components, and the user interface for this tool is awkward.

To use this tool, first select the tool, then enter the desired text in the 'Text' attribute in the attrutes panel at the bottom left. Then click anywhere on the circuit to insert a comment with that text.

In contrast to all other components in the current built-in libraries, comments can be placed anywhere on the canvas; they do not snap to the grid.

Attributes

Text
The text appearing in the comment. This can be edited by changing the attribute in the attribute table, or using the Text Tool directly the canvas.
Font
The font used for the comment.
Horizontal Alignment
Controls the horiztonal alignment, as well as where the text appears relative to the origin (i.e. where the mouse was clicked when creating the comment). "Left" uses left-alignment, and positions the text so its left edge is at the origin. "Right" uses right-alignment, and positions the text so its right edge is at the origin. "Center" horiztonally centers each line, and positions the text horizontally so the center is at the origin.
Vertical Alignment
Controls the where the text appears relative to the origin (i.e. where the mouse was clicked when creating the comment). "Base" positions the text so the origin is at the baseline of the first line of text. "Top" positions the text so the origin is at the top of first line of the text. "Bottom" positions the text so the origin is at the bottom of the last line of text. "Center" centers the text vertically at the origin.

FPGA Synthesis

Comments are silently during FPGA synthesis.
