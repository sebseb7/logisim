Menu Tool

Library: Mouse Tools
Introduced: 2.0 Beta 1

Behavior

The Menu Tool is the default action when right-clicking, or left clicking with Control held. This tool brings up a context-sensitive pop-up menu with actions, including some that might be relevent to the component that was clicked.

Hint: You can change the default mouse click behavior using the Mouse tab of the project options. This lets you change what each mouse button does.

The contents of the pop-up menu depends on what was clicked. Most components allow cut, copy, paste, delete, and similar actions. For user-defined components, the "View.." item will open up the corresponding subcircuit to reveal its inner state, in the same way as the magnifying glass when using the Poke Tool.

Attributes

None.
