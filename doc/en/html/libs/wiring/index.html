Wiring Library

The Wiring library includes basic elements that are needed for most circuits.

Splitter — join or split wires into buses
Pin — input to and output from a circuit
Probe — display values on wires
Tunnel — make connections without drawing wires
Clock — oscillates between zero and one
Constant — like an input, but with a fixed value
Bit Extender — zero-extend or sign-extend binary numbers
